# CLAUDE.md - Coach Web Portal

Portail web pour coachs permettant la gestion des élèves, programmes et plans nutritionnels.

## Commandes

```bash
cd coach-web && npm run dev    # Démarrer le serveur de dev
cd coach-web && npm run build  # Build production
cd coach-web && npm run lint   # Vérifier le code
```

## Conventions

- **Composants** : PascalCase (`StudentCard.tsx`)
- **Stores** : kebab-case (`students-store.ts`)
- **Pages** : kebab-case avec suffixe `-page` (`students-list-page.tsx`)
- **Modales** : kebab-case avec suffixe `-modal` (`add-student-modal.tsx`)
- **Alias imports** : `@/` pointe vers `src/`
